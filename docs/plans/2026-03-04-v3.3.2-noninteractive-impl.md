# v3.3.2 実装計画: -NonInteractive

作成日: 2026-03-04  
対象ブランチ: `feature/v3.3.2-noninteractive`

## 1. スコープ
- `-NonInteractive` を追加し、Y/N入力待ちを発生させない。
- `-NoRebootPrompt` を追加し、再起動確認を抑制可能にする。
- 終了コードを標準化し、CIやスケジューラ連携を容易にする。

## 2. CLI仕様（初版）
- `-NonInteractive`
  - 指定時: すべての確認プロンプトをポリシーで自動応答。
  - 未指定時: 現行どおり対話実行。
- `-NoRebootPrompt`
  - 指定時: 再起動確認を出さずに処理を継続。
  - `-NonInteractive` と併用時に有効。

## 3. 自動応答ポリシー
- 既定は安全寄りの `No`。
- 例外として、読み取り専用/情報表示のみの処理はそのまま実行可。
- 削除や更新など破壊的変更を伴う確認は明示ポリシーが無い限り `No`。

## 4. 終了コード規約（初版）
- `0`: すべて成功
- `1`: 一部失敗（継続実行は完了）
- `2`: 致命的エラーで中断
- `3`: 不正引数

## 5. 受け入れ基準
- 非対話環境で停止せず完走する。
- 既存対話モードの挙動を壊さない。
- PS5.1/PS7 の両方でテストを通過する。
