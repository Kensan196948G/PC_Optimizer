# v4.0 CLI 仕様

作成日: 2026-03-04  
対象スクリプト: `PC_Optimizer.ps1`

## 1. 引数一覧（v4.0）
- `-NonInteractive`
  - 対話入力を禁止し、自動応答ポリシーで継続
- `-WhatIf`
  - 変更を適用せず、実行予定のみ記録
- `-Tasks <string>`
  - 例: `"1,3,7"` 指定タスクのみ実行
- `-LogLevel <info|warn|error|debug>`
  - ログ粒度の制御
- `-NoRebootPrompt`
  - 再起動確認を表示しない
- `-FailureMode <continue|fail-fast>`
  - エラー発生時の継続/停止
- `-ExportDeletedPaths <csv|json>`
  - 削除候補一覧の事前出力
- `-OutputFormat <html|csv|json|all>`
  - レポート出力形式
- `-ConfigPath <string>`
  - 既定: `config/config.json`

## 2. 引数組み合わせルール
- `-WhatIf` は変更系処理を無効化する（更新適用/削除を実施しない）
- `-NoRebootPrompt` は `-NonInteractive` 併用時に有効
- `-ExportDeletedPaths` は `-WhatIf` 併用を推奨
- `-Tasks` 未指定時は全タスク対象

## 3. config/config.json（初版）
```json
{
  "logLevel": "INFO",
  "reportFormat": "HTML",
  "cleanupBrowserCache": true,
  "diskOptimize": true,
  "enableSecurityCheck": true
}
```

## 4. 終了コード標準
- `0`: 全処理成功
- `1`: 一部失敗（処理完走）
- `2`: 致命的エラー中断
- `3`: 不正引数
- `4`: 管理者権限不足

## 5. 互換性要件
- PS5.1/PS7 で同一引数セットが動作
- 既存実行（引数なし）との後方互換維持
