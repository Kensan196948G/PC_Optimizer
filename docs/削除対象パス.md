# 削除対象パス一覧

## 一時ファイル

| パス | 説明 |
|---|---|
| `C:\Windows\Temp\*` | Windows システム一時ファイル |
| `%TEMP%\*` | 現在のユーザー一時ファイル |
| `%USERPROFILE%\AppData\Local\Temp\*` | ユーザープロファイルの一時ファイル |

## Prefetch・キャッシュ

| パス | 説明 |
|---|---|
| `C:\Windows\Prefetch\*` | アプリケーション起動高速化キャッシュ |
| `C:\Windows\SoftwareDistribution\Download\*` | Windows Update のダウンロードキャッシュ |
| `C:\Windows\System32\DeliveryOptimization\Cache\*` | Delivery Optimization のキャッシュ（別パス） |
| `C:\Windows\SoftwareDistribution\DeliveryOptimization\Cache\*` | Delivery Optimization のキャッシュ（主パス） |

## エラーレポート・ログ

| パス | 説明 |
|---|---|
| `C:\ProgramData\Microsoft\Windows\WER\ReportArchive\*` | Windows エラー報告アーカイブ |
| `C:\ProgramData\Microsoft\Windows\WER\ReportQueue\*` | Windows エラー報告キュー |
| `C:\Windows\Logs\CBS\*` | Component-Based Servicing ログ（更新インストール記録） |

## アプリケーションキャッシュ

| パス | 説明 | 条件 |
|---|---|---|
| `%LOCALAPPDATA%\Microsoft\OneDrive\logs\*` | OneDrive ログ | パスが存在する場合のみ |
| `%APPDATA%\Microsoft\Teams\Cache\*` | Microsoft Teams キャッシュ | パスが存在する場合のみ |
| `%LOCALAPPDATA%\Microsoft\Office\16.0\OfficeFileCache\*` | Office ファイルキャッシュ | パスが存在する場合のみ |

## ブラウザキャッシュ

| パス | ブラウザ | 条件 |
|---|---|---|
| `%LOCALAPPDATA%\Google\Chrome\User Data\Default\Cache\*` | Chrome | パスが存在する場合のみ |
| `%LOCALAPPDATA%\Google\Chrome\User Data\Default\Code Cache\*` | Chrome | パスが存在する場合のみ |
| `%LOCALAPPDATA%\Google\Chrome\User Data\Default\GPUCache\*` | Chrome | パスが存在する場合のみ |
| `%LOCALAPPDATA%\Microsoft\Edge\User Data\Default\Cache\*` | Edge | パスが存在する場合のみ |
| `%LOCALAPPDATA%\Microsoft\Edge\User Data\Default\Code Cache\*` | Edge | パスが存在する場合のみ |
| `%LOCALAPPDATA%\Microsoft\Edge\User Data\Default\GPUCache\*` | Edge | パスが存在する場合のみ |
| `%APPDATA%\Mozilla\Firefox\Profiles\*\cache2\entries\*` | Firefox | 全プロファイル対応 |

## サムネイルキャッシュ

| パス | 説明 |
|---|---|
| `%LOCALAPPDATA%\Microsoft\Windows\Explorer\thumbcache_*.db` | エクスプローラーのサムネイル画像キャッシュ |

## Microsoft Store キャッシュ

| パス | 説明 | 条件 |
|---|---|---|
| `%LOCALAPPDATA%\Packages\Microsoft.WindowsStore_8wekyb3d8bbwe\LocalCache\*` | Store ローカルキャッシュ | パスが存在する場合のみ |
| `%LOCALAPPDATA%\Packages\Microsoft.WindowsStore_8wekyb3d8bbwe\LocalState\Cache\*` | Store 状態キャッシュ | パスが存在する場合のみ |

## ゴミ箱

`rundll32.exe shell32.dll,SHEmptyRecycleBinA 0` を使用してゴミ箱を空にします。
全ドライブのゴミ箱が対象です。

## DNS キャッシュ

`ipconfig /flushdns` を実行します。ファイルの削除ではなくメモリ上のキャッシュをクリアします。

## Windows イベントログ

`wevtutil cl Application` および `wevtutil cl System` でログをクリアします。
ファイル削除ではなく Windows イベントログサービス経由でのクリアです。
Security ログは対象外（特権要件と監査要件のため）。

---

## 削除・変更されないもの

- ユーザーのドキュメント・写真・動画
- インストール済みアプリケーションのデータ
- `C:\Windows` 配下のシステムファイル（Temp, Prefetch, Logs 以外）
- レジストリ
- Security イベントログ
- ブラウザのプロファイルデータ（ブックマーク・パスワード・Cookie など）
  ※ キャッシュのみを削除します

## 注意事項

- すべてのファイル削除は `-ErrorAction SilentlyContinue` で実行されます。
  ロック中のファイルや権限不足のファイルは静かにスキップされます。
- `C:\Windows\SoftwareDistribution\Download\*` は「Prefetch・更新キャッシュ削除」と
  「Windows Update キャッシュ削除」の両タスクで重複して実行されます（冪等）。
- SFC および DISM はファイルの「削除」ではなく「修復・診断」を行います。
