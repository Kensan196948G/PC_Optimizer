# リポジトリ運用方針

最終更新日: 2026-03-04

## 追跡対象（Git 管理に含める）
- `PC_Optimizer.ps1`
- `Run_PC_Optimizer.bat`
- `tests/` 配下のテストコード
- `docs/` 配下のドキュメント一式
- リリースノート（`docs/releases/`）

## 非追跡対象（Git 管理から除外）
- 実行時に生成されるログ/レポート（`logs/` 配下）
- ルート直下に一時的に生成されるログ（`PC_Optimizer_Log_*.txt`, `PC_Optimizer_Error_*.txt`）

## 補足
- 検証ログを証跡として共有する必要がある場合は、`docs/releases/` 配下へ要約を記載する。
- 生ログは原則コミットしない。
