## [4.0.1] - 2026-03-06

### 修正

- `Run_PC_Optimizer.bat` の行末コードを LF → CRLF に変換
  - `cmd.exe` は CRLF を必須とするため、LF のみでは正常動作しない問題を修正
  - `Test_Encoding.ps1` の BAT-1b テストが 2 件 FAIL していた問題を解消

### 追加

- `.gitattributes` を新規作成
  - `*.bat` / `*.cmd` を常に CRLF（`eol=crlf`）で管理するよう設定
  - `*.ps1` / `*.psm1` / `*.psd1` を CRLF で管理するよう設定
  - YAML / JSON / Markdown は LF で管理するよう設定
- `tools/Repair-CIEnvironment.ps1` を新規作成（CI 自動修復スクリプト）
  - CHECK 1: `Run_PC_Optimizer.bat` の CRLF 行末コードを自動検出・修復
  - CHECK 2: `PC_Optimizer.ps1` の UTF-8 BOM 付与を自動検出・修復
  - CHECK 3: `PC_Optimizer.ps1` の PowerShell 構文チェック（パースエラー検出）
  - CHECK 4: テストファイルの存在確認
  - CHECK 5: モジュールファイルの存在確認
  - CHECK 6: `config.json` の存在・JSON 形式確認
- `.github/workflows/ci.yml` に自動修復ジョブ (`auto-repair`) を追加
  - `test-powershell-51` / `test-powershell-7` の両ジョブは `auto-repair` ジョブ成功後に実行
  - 各テストジョブの事前に `Repair-CIEnvironment.ps1` を再実行して環境整合性を保証
  - CI サマリージョブ (`ci-summary`) を追加して全ジョブ結果を一覧表示
  - トリガーに `copilot/**` ブランチを追加

### ドキュメント

- 本ファイル（`docs/変更履歴.md`）を更新

## [3.0.0] - 2026-02-19

### 追加

- **全 UI の日本語化**
  - `PC_Optimizer.ps1` の全コンソール表示・ログメッセージ・ステップ名を日本語化
  - `Run_PC_Optimizer.bat` の全コメント・echo メッセージを日本語化
- **ログ出力先を `logs/` サブフォルダに変更**（実行のたびに自動作成）
- **ブラウザキャッシュの削除**（Google Chrome / Microsoft Edge / Mozilla Firefox 全プロファイル対応）
- **サムネイルキャッシュの削除**（`thumbcache_*.db`）
- **Microsoft Store キャッシュのクリア**（LocalCache / LocalState\Cache）
- **Windows イベントログのクリア**（Application / System、`wevtutil cl` 使用）
- **システムファイルの整合性チェック・修復（SFC）**（`sfc /scannow`）
- **Windows コンポーネントストアの診断（DISM）**（`DISM /Online /Cleanup-Image /ScanHealth`）
- **電源プランの最適化**
  - ノートPC（バッテリー検出時）→「バランス」（GUID: 381b4222-...）
  - デスクトップ PC → 「高パフォーマンス」（GUID: 8c5e7fda-...）、未登録時は自動追加
- **実行前後のディスク空き容量比較表示**（解放容量を GB 単位で表示）
- **文字コード検証テスト（`Test_Encoding.ps1`）** の大幅拡充
  - BAT: CRLF 行末コードチェック追加
  - PS1: 新規 27 日本語 UI 文字列の存在確認、PowerShell 構文チェック追加

### 修正

- **`Run_PC_Optimizer.bat` の行末コードを CRLF に統一**
  （LF のみだと `cmd.exe` の `if (...) (...)` ブロック解析が失敗するバグを修正）
- `Run_PC_Optimizer.bat` の Shift-JIS (CP932) + CRLF 書き込みに `[System.IO.File]::WriteAllBytes` を使用するよう変更

### ドキュメント

- `docs/` フォルダ内の全ファイル名を日本語に変更
- 全ドキュメントを新機能・日本語化に合わせて更新
