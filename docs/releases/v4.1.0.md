# v4.1.0 リリースノート

**リリース日**: 2026-03-05
**タグ**: `v4.1.0`

## 概要

v4.0.1 Hardening に続く機能拡張リリース。Slack/Teams 通知・Chart.js スコア推移グラフ・AgentSDK テストフレームワークを追加。

## 主な変更

### 追加

- **`modules/Notification.psm1`** 新規作成
  - `Send-SlackNotification`: Slack Incoming Webhook へスコア・評価・推奨アクションを POST（スコアに応じた色分け）
  - `Send-TeamsNotification`: Teams MessageCard を POST
  - `Send-McpProviderNotification`: mcpProviders 配列の enabled=true エントリを自動 dispatch
  - PS7 / PS5.1 分岐（PS5.1 は Invoke-WebRequest + UTF-8 デコード）

- **`modules/Report.psm1`** Chart.js グラフ追加
  - `Update-ScoreHistory` 関数: `reports/score_history.json` にスコアを最大 30 件保持
  - HTML レポートにスコア推移折れ線グラフ（scoreTrendChart）追加
  - HTML レポートにカテゴリ別レーダーチャート（categoryRadarChart）追加

- **`tests/AgentSDK.Tests.ps1`** 新規作成
  - AT-01〜AT-10 が全 PASS
  - Test-AgentPluginContext / New-AgentPluginResult / Test-AgentPluginResultSchema / Get-AgentCollectorResult 検証

- **`docs/エージェント開発ガイド.md`** 新規作成
  - Agent DAG の役割説明・実装例・登録方法・テスト方法

- CI: Pester カバレッジ閾値 20% → 30% に引き上げ

## 動作確認済み環境

- Windows 11 24H2 / PowerShell 7.4.x
- Windows 10 21H2 / PowerShell 5.1

## アップグレード手順

ツールフォルダを最新版に差し替えてください（設定 `config/config.json` は上書き不要）。
