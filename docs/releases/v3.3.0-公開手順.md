# v3.3.0 公開手順

このリポジトリは現在 `git remote` 未設定のため、push 実行前にリモート登録が必要。

## 1. リモート登録
```bash
git remote add origin <REMOTE_URL>
git remote -v
```

## 2. ブランチとタグの公開
```bash
git push -u origin release/v3.3.0
git push origin v3.3.0
```

## 3. master 反映済み内容の公開（必要時）
```bash
git push origin master
```

## 4. 公開確認
```bash
git ls-remote --heads origin release/v3.3.0
git ls-remote --tags origin v3.3.0
```
