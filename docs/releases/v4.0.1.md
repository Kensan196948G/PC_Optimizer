# v4.0.1 リリースノート

リリース日: 2026-03-05  
ブランチ: `release/v4.0.0`  
前バージョン: v4.0.0  
コミット: `828c3c6`

## 概要

v4.0.0 の CLI 運用安定化を目的とした Hardening リリース。  
入力バリデーション異常系テストの大幅拡充・CI 診断ログ改善・fail-fast レポート実検証に加え、
コードレビューで検出された **セキュリティ脆弱性 2 件 / 規約違反 1 件 / バグ 3 件** を修正。

## 変更点

### セキュリティ修正

| ファイル | 深刻度 | 内容 |
|---|---|---|
| `modules/Orchestration.psm1` | Critical | Hook `command` 型で `Invoke-Expression` による任意コード実行を可能にしていた脆弱性を廃止 |
| `PC_Optimizer.ps1` (`Clear-DirContents`) | Critical | `cmd.exe` + 非サニタイズパスによるコマンドインジェクションを `Remove-Item -LiteralPath` に置き換え |
| `modules/*.psm1` (全4モジュール) | High | ファイル書き込みで `-Encoding utf8` をハードコードしていた規約違反を修正。`$script:_enc` で PS バージョン分岐 |

### バグ修正

| ファイル | 内容 |
|---|---|
| `modules/Advanced.psm1` | PS5.1 で AI 診断レスポンスが文字化けするバグを修正（`Invoke-WebRequest` + 明示的 UTF-8 デコード） |
| `PC_Optimizer.ps1` (`Get-SystemDriveFreeGB`) | `$null` 比較の順序誤り（配列比較リスク）を `$null -ne $prop` に修正 |
| `PC_Optimizer.ps1` (電源プラン最適化) | `powercfg` の `$LASTEXITCODE` を即時キャプチャするよう修正 |
| `PC_Optimizer.ps1` (ディスク最適化) | `defrag` 非ゼロ終了コード時にログ出力を追加 |

### テスト拡充

| テストファイル | 追加ケース | 内容 |
|---|---|---|
| `tests/Test_TasksValidation.ps1` | TV-08〜TV-17 | Tasks 下限/上限超過・重複範囲・逆順・正常系 |
| `tests/Test_CLIModes.ps1` | EP-01〜EP-06 | ExportDeletedPathsPath 予約語・空文字・正常系 |
| `tests/Test_ReportSchema.ps1` | RS-17〜RS-23 | fail-fast 実実行 JSON レポート検証 |

### CI 改善

- 全テストジョブ（PS5.1 / PS7）に失敗時の診断メタ情報収集ステップを追加
- 収集情報: PS バージョン・OS ビルド・ディスク空き容量・失敗テストログ名
- 生成ファイル: `test-results/diag_meta_ps51.json` / `test-results/diag_meta_ps7.json`
- アーティファクト: `diag-meta-ps51` / `diag-meta-ps7`（14日保持）
