# 関数リファレンス

## Write-Log

```powershell
Write-Log($msg)
```

メインログファイル（`logs/PC_Optimizer_Log_YYYYMMDDHHMM.txt`）に 1 行追記します。

| パラメータ | 型 | 説明 |
|---|---|---|
| `$msg` | string | ログに書き込む文字列 |

タイムスタンプは呼び出し側で付与します。

---

## Write-ErrorLog

```powershell
Write-ErrorLog($msg)
```

エラーログファイル（`logs/PC_Optimizer_Error_YYYYMMDDHHMM.txt`）に追記します。
先頭行に `[ERROR] yyyy/MM/dd HH:mm:ss` のタイムスタンプを自動挿入します。

| パラメータ | 型 | 説明 |
|---|---|---|
| `$msg` | string | エラー詳細文字列（`catch` ブロックの `$_` を推奨） |

---

## Show

```powershell
Show($msg, $color = "White")
```

色付きでコンソールに出力します。ログファイルには書き込みません。

| パラメータ | 型 | デフォルト | 説明 |
|---|---|---|---|
| `$msg` | string | — | 表示する文字列 |
| `$color` | string | `"White"` | `Write-Host` の `-ForegroundColor` に渡す色名 |

使用例：

```powershell
Show "処理完了" Green
Show "警告: SSD 未検出" DarkGray
Show "エラーが発生しました" Red
```

---

## Progress-Bar

```powershell
Progress-Bar($msg, $percent)
```

コンソールにプログレスバーと説明を表示します。

| パラメータ | 型 | 説明 |
|---|---|---|
| `$msg` | string | バーの右に表示するメッセージ |
| `$percent` | int | 進捗率（0 〜 100）。10% 単位で描画される |

---

## Try-Step

```powershell
Try-Step($desc, [ScriptBlock]$action)
```

最適化タスクをラップし、開始/完了/失敗のログ記録とプログレスバー表示を自動で行います。

| パラメータ | 型 | 説明 |
|---|---|---|
| `$desc` | string | タスクの説明（ログとバーに表示） |
| `$action` | ScriptBlock | 実行するタスクのコードブロック |

---

## Test-PendingReboot

```powershell
[bool] Test-PendingReboot()
```

## Run-WindowsUpdate

```powershell
Run-WindowsUpdate()
```

COM API で利用可能な更新を検索・一覧表示し、ユーザーの確認を得てから適用します。

スタートアップ登録アプリと不要サービスの稼働状況を読み取り専用で表示します。

## modules/Report.psm1

### Update-ScoreHistory

```powershell
Update-ScoreHistory -ReportData <pscustomobject> [-HistoryPath <string>]
```

`HistoryPath` に実行ごとのスコアを蓄積します（最大 30 件）。

| パラメータ | 型 | 説明 |
|---|---|---|
| `$ReportData` | pscustomobject | 診断結果（score, cpuScore, memoryScore 等を含む） |
| `$HistoryPath` | string | score_history.json のパス（省略時は処理スキップ） |

---

### Export-OptimizerReport

```powershell
Export-OptimizerReport -ReportData <pscustomobject> -Format <string> -Path <string> [-UseLocalChartJs] [-ChartJsScriptPath <string>]
```

レポートデータを指定フォーマット（html/csv/json）でエクスポートします。

| パラメータ | 型 | 説明 |
|---|---|---|
| `$ReportData` | pscustomobject | レポートデータ（New-OptimizerReportData の出力） |
| `$Format` | string | 出力形式（`html` / `csv` / `json`） |
| `$Path` | string | 出力先ファイルパス |
| `-UseLocalChartJs` | switch | ローカル Chart.js を使用するか |
| `$ChartJsScriptPath` | string | Chart.js のローカル相対パス |

---

## modules/Advanced.psm1

### Invoke-AIDiagnosis

```powershell

```

ヘルススコアとシステム状態をもとに AI 診断（Anthropic Claude API）またはローカルルールエンジンで診断結果を返します。

|---|---|---|
| `Source` | string | `"Anthropic"` または `"LocalRuleEngine"` |
| `FallbackReason` | string | フォールバック時の理由 |
| `Confidence` | string | 診断の信頼度 |
| `DataTimestamp` | string | 診断実施時刻 |
| `InputMetrics` | pscustomobject | 診断に使用した入力メトリクス（ScoreInput が安全にアクセスされます） |
