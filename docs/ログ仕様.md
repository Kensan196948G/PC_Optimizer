# ログファイル仕様

## 出力場所

スクリプト実行に関連するファイルは以下のフォルダに出力されます:

| フォルダ | 内容 | 自動生成 |
|---|---|---|
| `logs\` | 通常ログ・エラーログ | ✅ |
| `logs\siem\` | SIEM 出力ファイル（オプション機能有効時のみ） | ✅ |
| `reports\` | HTML/CSV/JSON レポート・スコア履歴 | ✅ |
| `reports\mcp\` | MCP プロバイダー実行ログ | ✅ |

## ファイル命名規則

```
logs\PC_Optimizer_Log_YYYYMMDDHHMM.txt       ← 通常ログ
logs\PC_Optimizer_Error_YYYYMMDDHHMM.txt     ← エラーログ（エラー発生時のみ生成）
```

`YYYYMMDDHHMM` はスクリプト起動時刻です（例: `202602191200`）。

## タイムスタンプ形式

| 箇所 | 形式 | 例 |
|---|---|---|
| ファイル名 | `yyyyMMddHHmm` | `202602191200` |
| ログ本文（開始・終了） | `yyyy/MM/dd HH:mm:ss` | `2026/02/19 12:00:00` |
| タスク行 | `HH:mm:ss` | `12:00:01` |

## エンコーディング

PowerShell バージョンを自動検出し、適切なエンコーディングで書き込みます。

```powershell
$logEncoding = if ($isPS7Plus) { 'utf8NoBOM' } else { 'UTF8' }
Add-Content -Path $logPath -Value $msg -Encoding $logEncoding
```

| PowerShell | エンコーディング |
|---|---|
| 5.1 | UTF-8 with BOM（`UTF8`） |
| 7.x | UTF-8 without BOM（`utf8NoBOM`） |

## SIEM 出力（オプション機能）

config.json で `hooks.siem.enabled: true` に設定すると、Hook 実行ログを SIEM 対応フォーマットで出力します。

### 出力場所・命名規則

```
logs\siem\siem_YYYYMMDD.jsonl   ← JSONL 形式（デフォルト）
logs\siem\siem_YYYYMMDD.cef     ← CEF 形式
logs\siem\siem_YYYYMMDD.leef    ← LEEF 形式
```

### JSONL 形式の例

```json
{"timestamp":"2026-03-05T18:00:00","severity":3,"action":"hook.on_start","status":"OK","runId":"abc123","hostName":"DESKTOP-XXXX"}
{"timestamp":"2026-03-05T18:05:00","severity":7,"action":"hook.on_error","status":"NG","runId":"abc123","hostName":"DESKTOP-XXXX","error":"タスク失敗"}
```

### severity 値の対応

| severity | 意味 |
|---|---|
| 3 | 成功（INFO） |
| 7 | 失敗/警告（ERROR） |

### 有効化方法

config.json:
```json
"hooks": {
  "siem": {
    "enabled": true,
    "format": "jsonl",
    "outputPath": "logs\\siem"
  }
}
```

## スコア履歴ファイル

Agent Teams モードで実行した場合、AI 診断スコアが `reports\score_history.json` に蓄積されます（最大 30 件）。

```json
[
  { "timestamp": "2026-03-05T18:00:00", "overallScore": 86, "evaluation": "良好" },
  { "timestamp": "2026-03-04T10:00:00", "overallScore": 82, "evaluation": "普通" }
]
```

このデータは HTML レポートの Chart.js スコア推移グラフと レーダーチャートに使用されます。

## ログのローテーション

自動ローテーション機能はありません。
実行のたびに新しいファイルが生成されるため、定期的に古いログファイルを削除してください。

不要なログを一括削除するコマンド例：

```powershell
# 30日以上前のログを削除
Get-ChildItem -Path "C:\path\to\PC_Optimizer\logs" -Filter "PC_Optimizer_*.txt" |
    Where-Object { $_.LastWriteTime -lt (Get-Date).AddDays(-30) } |
    Remove-Item -Force
```
